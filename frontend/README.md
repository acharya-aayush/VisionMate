# VisionMate Frontend

React app for hand/object detection and voice feedback. Sends images to backend for face recognition.

Stack:
- React, Vite, TypeScript
- Tailwind CSS
- TensorFlow.js (hand/object detection)
- TanStack Query
- shadcn/ui, Radix UI
- React Router

How to run:
1. npm install
2. npm run dev

This is just the frontend. Backend required for face recognition.
